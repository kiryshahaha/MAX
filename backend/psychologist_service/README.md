Микросервис для бронирования и просмотра записей к психологам.  
Сервис предоставляет API для пользователей и психологов, включая проверку расписания и занятости.

## Основной функционал

1. Создание записи к психологу (для пользователя)
   - Пользователь выбирает психолога и время записи.
   - Система проверяет:
     - Психолог работает в выбранный день.
     - Время кратно целому часу (например, 09:00, 14:00).
     - Время ещё свободно.
   - Если всё верно, запись сохраняется в базе Supabase.

2. Просмотр своих записей (для пользователя)
   - Пользователь может получить список всех своих записей.
   - API возвращает дату, время, имя психолога и заметки.

3. Просмотр занятости психолога (для психолога)
   - Психолог может увидеть своё расписание:
     - Когда у него есть записи.
     - Свободные часы для записи.

4. Получение доступных слотов (для удобства фронтенда)
   - Сервис возвращает свободные часовые слоты на выбранный день для конкретного психолога.


Для записи доступны 2 психолога с расписанием:
"Клепов Дмитрий Олегович"
        "Tuesday": [(time(16, 0), time(20, 0))],
        "Thursday": [(time(16, 0), time(20, 0))],
        "Saturday": [(time(11, 0), time(16, 0))]

    "Кашкина Лариса Владимировна"
        "Monday": [(time(10, 0), time(14, 0))],
